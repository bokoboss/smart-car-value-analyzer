วิธีสร้าง API Key สำหรับ Google AI

  1. ไปที่เว็บไซต์ Google AI Studio และล็อกอินด้วยบัญชี Google
  2. คลิกปุ่ม "Get API key" (อาจอยู่ที่เมนูด้านซ้าย)
